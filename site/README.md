# site/ — GS1 Norway nettside

`index.html` er tilbudssiden som deles med GS1. Ingen innlogging — en unik,
delbar URL (samme mønster som revised.v2.lynxeye.threesixty1.com).

## Deploy-mønster (unik URL)

Følg samme oppsett som Lynxeye-referansen:
- Subdomene under threesixty1.com, f.eks. `gs1.threesixty1.com` eller en
  ugjettbar variant for "soft privacy" (`v1.gs1norway.threesixty1.com`).
- Statisk HTML — kan hostes hvor som helst (Vercel, Netlify, egen server).
- Fordi den er statisk og din egen, kan du oppdatere og pushe endringer raskt
  uten at GS1 trenger å gjøre noe.
